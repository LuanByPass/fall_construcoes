# 🏗️ FALL Construções - Sistema de Estoque e Vendas

Sistema completo para casa de construção com:
- Controle de estoque com código de barras
- PDV (Ponto de Venda)
- Emissão de comprovantes
- Relatórios financeiros
- Gestão de clientes

## 🚀 Como Usar

1. Instale MySQL e execute o script `fall_database.sql`
2. python -m venv .venv `Caso o ambiente nao esteja instalado`
3. `pip install -r requirements.txt`
4. `python main.py`
5. Login: `luan` / `1014`

## 📦 Funcionalidades

- ✅ Leitura de código de barras
- ✅ Vendas com desconto
- ✅ Múltiplas formas de pagamento
- ✅ Relatórios de estoque e vendas
- ✅ Ranking de produtos
- ✅ Gestão de clientes
- ✅ Console de logs em tempo real
